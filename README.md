# dbcompare
数据库结构比对工具
使用方式:

A.比对增量下载：

针对线上环境:1.表结构比较-》2.输入比对库(可选择时间点),点击比对-》3.创建脚本(查看压缩文件里readme.txt说明是否有漏)

针对测试环境:1.表结构比较-》2.输入比对库(可选择时间点),点击比对-》3.创建脚本-》4.执行后再比对如果还有差异,点击创建DDL补丁脚本
